# Устранение неполадок

## Требования к окружению

Для локальной сборки и CI необходимо:

- **macOS** — 12+
- **Xcode** — 14+ (рекомендуется 16.x)
- **Целевая версия iOS** — 16.0 (из Podfile) 
- **Target Version** - 77
- **Ruby** — 3.3+
- **Bundler** — для установки гемов
- **CocoaPods** — 1.16+
